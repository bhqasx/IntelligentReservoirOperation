# PopHistory 数据分析工具

这个工具用于分析NSGA-III优化算法的历史数据文件 `PopHistory.json`，生成四个子图来可视化优化过程。

## 功能特点

1. **自动数据读取**：从 `PopHistory.json` 文件中读取多代优化数据
2. **统计分析**：计算每代目标函数的均值、最小值、最大值
3. **趋势分析**：分析优化过程中目标函数的变化趋势
4. **可视化展示**：生成四个专业的子图

## 生成的四个子图

### 子图1：目标函数1 vs 目标函数2
- 显示第一代和最后一代所有个体的第1、2目标函数值
- 蓝色圆点：第一代数据
- 红色方块：最后一代数据

### 子图2：目标函数1 vs 目标函数3
- 显示第一代和最后一代所有个体的第1、3目标函数值
- 蓝色圆点：第一代数据
- 红色方块：最后一代数据

### 子图3：目标函数2 vs 目标函数3
- 显示第一代和最后一代所有个体的第2、3目标函数值
- 蓝色圆点：第一代数据
- 红色方块：最后一代数据

### 子图4：目标函数平均值趋势
- 横坐标：代数
- 纵坐标：目标函数平均值
- 三条曲线分别表示三个目标函数随代数的变化

## 使用方法

### 1. 确保环境配置
```bash
# 激活虚拟环境
D:/MyGitRepo/IntelligentReservoirOperation/Optimize/.venv/Scripts/activate

# 安装依赖包
pip install numpy matplotlib
```

### 2. 运行分析脚本
```bash
python analyze_pop_history.py
```

### 3. 查看结果
- 控制台会显示统计信息
- 会弹出matplotlib图形窗口显示四个子图
- 可以与图形进行交互（缩放、旋转等）

## 数据格式要求

`PopHistory.json` 文件结构：
```json
{
  "generations": {
    "1": {
      "obj": [[obj1, obj2, obj3], [obj1, obj2, obj3], ...],
      "P_plans_SMX": [...],
      "P_plans_XLD": [...],
      "ConstraintViolation": [...]
    },
    "2": {...},
    ...
  }
}
```

## 注意事项

1. 确保 `PopHistory.json` 文件与脚本在同一目录
2. 如果数据量很大，可能需要等待一段时间来生成图形
3. 图形窗口支持交互操作，可以旋转3D视图、缩放等
4. 如果遇到中文显示问题，请确保系统安装了中文字体

## 依赖包

- numpy >= 1.20.0
- matplotlib >= 3.5.0

## 故障排除

1. **ModuleNotFoundError**: 确保安装了所有依赖包
2. **FileNotFoundError**: 检查 `PopHistory.json` 文件是否存在
3. **图形不显示**: 尝试修改matplotlib后端或使用不同的显示设置
